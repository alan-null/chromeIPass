## 2. Installation

### 2.1 Requirements
- [KeePassXC](https://keepassxc.org/) v2.1.1 or higher

OR
- [KeePass](http://keepass.info) v2.17 or higher
- [KeePassHttp](https://github.com/alan-null/keepasshttp) v1.0.7 or higher (recommended v1.4 or higher)

It is recommended to disable the built-in Chrome password management when using this extension.

### 2.2 Installation
1. Unlock your KeePass database.
2. Install from the [Chrome Web Store](https://chromewebstore.google.com/detail/chromeipass/bocdfmfhfcobbagfcbimemeclglilcbk).
3. New browser icon appears:<br />
![browser-icon](https://raw.github.com/alan-null/chromeIPass/master/documentation/images/cip-browser-icon.png)
4. Click icon and connect to KeePassHttp:<br />
[<img src="https://raw.github.com/alan-null/chromeIPass/master/documentation/images/cip-popup-connect.png" alt="popup-connect" width="200px" />](https://raw.github.com/alan-null/chromeIPass/master/documentation/images/cip-popup-connect.png)
5. Name the pairing request in KeePassHttp (e.g. "Chrome on my main computer").<br />
[<img src="https://raw.github.com/alan-null/chromeIPass/master/documentation/images/keepass-association-key.png" alt="keepass-association-key" width="200px" />](https://raw.github.com/alan-null/chromeIPass/master/documentation/images/keepass-association-key.png)
6. Re-open popup to verify normal state:<br />
[<img src="https://raw.github.com/alan-null/chromeIPass/master/documentation/images/cip-popup-normal.png" alt="popup-normal" width="200px" />](https://raw.github.com/alan-null/chromeIPass/master/documentation/images/cip-popup-normal.png)
7. Reload current page.
8. Open settings and adjust.
